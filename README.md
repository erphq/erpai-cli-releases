# ERP·AI CLI

AI-powered command-line interface for your ERP data.

## Quick Install

```sh
curl -fsSL https://raw.githubusercontent.com/ERPdotAI/erpai-cli-releases/main/install.sh | \
  sudo env REPO=ERPdotAI/erpai-cli-releases VERSION=v0.1.5 sh
```

## Getting Started

1. **Install the CLI** - Run the install command above
2. **Log in** - `erpai login`
3. **Start chatting** - `erpai chat`

## Website Development

This repo also contains the landing page website.

```sh
npm install
npm run dev
```

## Requirements

- macOS (Apple Silicon or Intel)
- Linux (x64)
- Windows (x64)
- ERP·AI account
