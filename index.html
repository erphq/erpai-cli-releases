Quick Install

curl -fsSL https://raw.githubusercontent.com/ERPdotAI/erpai-cli-releases/main/install.sh | \
  sudo env REPO=ERPdotAI/erpai-cli-releases VERSION=v0.1.5 sh

Getting Started

1

Install the CLI

Run the install command above. The binary will be installed to /usr/local/bin

2

Log in to your account

Authenticate with your ERP·AI credentials

erpai login
3

Start chatting

Query your ERP data using natural language

erpai chat
